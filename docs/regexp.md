## 量词

| 字符  | 说明           |
| ----- | -------------- |
| ?     | 重复零次或一次 |
| *     | 重复零次或多次 |
| +     | 重复一次或多次 |
| {n}   | 重复n次        |
| {n,}  | 重复n次以上    |
| {m,n} | 重复m到n次     |
